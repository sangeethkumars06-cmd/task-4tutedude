# Task 4 – CSS Units, Box Model & Fonts

## Overview

This project is part of the MERN Stack Web Development course. The objective of this task is to understand and apply CSS units, the CSS box model, viewport units, and font styling while creating a simple webpage consisting of an image and a button.

## Features

* Semantic HTML5 structure
* External CSS stylesheet
* Image displayed inside a container
* Gradient-styled button
* Proper use of CSS Box Model
* Viewport (`vw` and `vh`) units for responsive spacing
* Equal left and right spacing of 25%
* Top and bottom spacing of 10%
* Page fits within a single screen without scrolling

## Technologies Used

* HTML5
* CSS3

## Project Structure

```text
Task4/
├── index.html
├── style.css
└── README.md
```

## How to Run

1. Clone or download this repository.
2. Open the project folder.
3. Ensure `index.html` and `style.css` are in the same directory.
4. Open `index.html` in any modern web browser.

## Concepts Practiced

* CSS Units (`vw`, `vh`, `%`)
* CSS Box Model
* `box-sizing` property
* Font styling
* Background gradients
* Responsive spacing using viewport units
